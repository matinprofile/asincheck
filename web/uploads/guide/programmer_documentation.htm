As you know, Yii is a MVC framework. So our project is contained by three part. Model, View and Controller. These parts have own folders. In controllers, models and views folder, there is another main folder called amazon. this folder consist of some php script file.

 

Installation:

Note : We assume that Yii was installed on your host.

·         Copy amazon folders from controllers, models and views from module to yii controllers, models and views folder.

·         Set your default controller to “amazon/analyzer” by adding following code on “root/config/web.php”:

'defaultRoute' => 'amazon/analyzer',

 

·         If you want to use this module in your site, you have two ways:

o   Refer to default controller from your site like this:

http://[your_domain]/[yii_root]/web/index.php?r=amazon/analyzer
